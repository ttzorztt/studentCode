什么是颅面复原技术
━━

 

 

 
       颅面复原技术即三维颅面鉴定技术，属于国际法医学个体识别领域前沿领域。计算机辅助的颅面复面貌原技术是对三维数字化颅面数据，应用人类学关于颅面面貌的软组织厚度数据，采用计算机技术来生成该颅面对应的面貌，使得生成的面貌尽可能的符合该颅面的原面貌特征。它是建立在科学计算可视化、人工智能、图像处理与理解及法医人类学领域知识基础上，结合了计算机、法学、计算医学、系统工程、数学、艺术等多方面知识的面向身份认证的科学技术。计算机辅助的颅面面貌复原方法作为信息科学新技术在颅面复原中的全新应用，相对于传统人工复原方法更具客观性和高效方面的优势，成为近年来国内外研究热点，在医学、刑侦、考古等领域具有广泛的应用。

颅面复原技术类型
━━

 

 

        颅面面貌复原技术分为手工颅面复原和计算机辅助的三维颅面复原。

        手工复原术是基于面部雕塑法，不仅要求技术人员懂得雕塑手法又要懂得解剖原理，换句话说复原面貌是一件技术性很强的工作，在手工复原过程中耗时长,主观性强,还缺乏质感，复原效果不理想，尤其在刑侦方面的应用收到了很大的限制。如下列左图所示。

       而计算机辅助的颅面复原完全是由科学数据确定的，不仅耗时短，而且准确率也是非常有保障的。科研人员对颅面进行数字化，科研人员通过使用手持式激光三维扫描仪对颅面进行扫描，从而获取了准确反映颅面表面几何形态的三维点云和表面颜色数据，经过后期封装和虚拟拼接，完成颅面的三维建模，并以此为基础对该颅面生前样貌进行复原。如右图所示。

        例如唐代公主李倕的颅面，在运回研究所的时候用石膏进行固定，但是在进行研究的时候遇到了一个难题，颅面和头冠被石膏连在了一起,如果用手工复原的方法肯定会浪费很多时间或许还会对颅面或者头冠造成一些损伤，但是计算机辅助的颅面复原就不一样,使用ct扫描，因为ct机并不会扫上石膏,只会取颅面，比如颅面肌肉，土和骨头都会分得很清楚。通常几个小时甚至是几十个小时才能完成的工作，计算机辅助的颅面复原技术只需要几十分钟就能完成，为我们的考古工作和刑侦工作提供了极大地便利。

颅面复原技术实现方法
━━

 

 
       计算机辅助的三维颅面复原技术一般分为三维颅面重建、软组织添加、五官组合修饰三个阶段。
       在研究过程中，可以用三坐标测量仪或者3D扫描仪直接扫描颅面，然后再计算机重建三维数字化颅面图像；也可以利用X射线、共振以及CT等医疗检测设备，由二维断层图像重建三维颅面图像。数字化三维颅面建模之后，要在数字化颅面上添加软组织，可以把组织的添加分为两种:一种是通过颅面的变形用所谓硬组织充填软组织的空间，另一种是确定软组织的厚度后定量的充填软组织，常用的有模板法和三维凸包和分区法。之后基于一般几何模型变形的人脸建模方法，该方法使用不同角度的人脸图像提取人脸特征点，并使用视觉方法恢复这些点的三维位置，然后修改一般人脸模型，最后进行人脸图像的纹理映射得到特定的三维人脸。
