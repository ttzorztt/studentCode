考古领域
━━━━
 

 

技术

 

考古方面的颅面复原主要采用的是计算机辅助的三维颅面面貌复原技术。

  

 

应用

 

计算机辅助的颅面复原技术中计算机辅助三维复原系统的应用。

  

 

过程

 

一般分为三维颅面重建、软组织添加、五官组合修饰三个阶段。

  

 

实例

 

借助现代科技技术手段唐朝公主李倕容貌恢复，再现风采。
考古技术
━━

 

 

       从头颅样本中恢复出人逼真的面部模型，在考古学中对古尸面容的复原、在刑事案件中对受害者的人脸识别等应用领域都具有非常重要的作用，其中计算机辅助的颅面复原技术发挥了很大的作用，采用见算计来进行颅面复原主要有两种方法：基于特征点的变形方法和基于可变形模板的方法前者在研究了不同人种的头颅和皮肤关系之后，后者着重研究了附着在头颅上的特征点和与脸部软组织的深度关系，然后以这些控制点变形生成人的头部模型。基于这样的原理，可变形模板也可以采用活动肌肉与头颅面自身形体比例法的关系来重建三维人面模型。

 

 

 

考古应用
━━

 

 

       计算机辅助的颅古面貌复原作为计算机图形学领域的一个新兴领域，得到了人们的广泛关注。从人类学研究到法医学应用，从古人类复原到刑事侦查都对面貌复原提出了应用要求。不同的用途对基于颅面的面貌复原系统要求是不同的，流程也不尽相同，但是对真实感和可靠性的要求是共通的，其中最显著的应用就是注明的古人面貌复原：马王堆的主人辛追的面貌复原、明宁王妃的复原、唐朝公主李倕的面貌复原，无不体现了基于颅面的面貌复原技术的魅力。历史学家们总在不断追寻更加成熟有效的系统，希望利用最新的学术研究成果更加准确快速的完成历史人物的面貌复原工作。

 

 

 

考古过程
━━

 

 

 

 

       重建三维颅面：在研究过程中，可以用三坐标测量仪或3D扫描仪直接扫描颅面，然后再计算机重建三维数字化颅面图像；也可以利用X射线、磁核共振以及CT等医疗检测设备，由二维断层图像重建三维颅面图像。比较经典的方法有：①用径向基函数从不完整的扫描数据生成连续网格;②基于CT等值面数据生成颅面曲面;③用傅里叶级数拟合CT图像提取颅面边缘轮廓曲线;④基于形状的利用数学形态学算子进行骨架提取的插值算法。

        软组织添加：数字化三维颅面建模之后，要在数字化颅面上添加软组织，可以把软组织的添加分为两种:一种是通过颅面的变形用所谓硬组织充填软组织的空间，另一种是确定软组织的厚度后定量的充填软组织，常用的有：①模板法。这种方法建立在数据库的基础上,用数据库存放大量的通过扫描各种人物面孔获得的数字化的面部模板。复原时,根据颅面的特点选择适当的面部模板,把它像面罩一样覆盖到颅面上,同时,对面模调整变形使之于待复原颅面的标志点的位置匹配。这种方法由vanezis于2000年提出,由该方法得到的复原头像一般是不完整的,头顶和下巴经常缺失。②三维凸包和分区法。改法结合了凸包技术、模板技术和变形技术,是充分发挥了各方法优点的一种新方法。采用三维包裹技术(quick一hull算法)包裹整个颅面。软组织厚度数据可根据颅面的子区域大小进行比例伸缩,合理的适应待复原颅面结构。该方法可以极大地减少对专家的依赖,是重建效果基本真实的一种方法。

        五官复原：之后基于一般几何模型变形的人脸建模方法，该方法使用不同角度的人脸图像提取人脸特征点，并使用视觉方法恢复这些点的三维位置，然后修改一般人脸模型，最后进行人脸图像的纹理映射得到特定的三维人脸，其中典型的由Guenter等人的人脸造型技术。

 

 

考古实例
━━

 

 

        2001年11月至2003年6月,考古人员在西安发现李倕墓,当时棺木已朽,尸骨尚存。考古专家发现了一件完整的头冠,这顶头冠在头颅遗骸上,虽然已经散乱,可头冠几乎用尽了唐代可能用到的所有装饰材料:绿松石、琥珀、珍珠、红宝石、贝壳、玛瑙、金银铜铁等,很多金饰件下还有翡翠鸟鲜艳的蓝色羽毛……色彩绚烂,极尽奢华。2010年2月,德国巴伐利亚博物馆和陕西省考古研究院的工作人员花了三年时间，完成了头冠的物理复原，构建出头冠的三维数字化模型，虚拟修复后的头冠由5000多个模块、共14个部件组成，是目前世界上唯一一件近乎完整的、经数字化复原的唐代公主冠饰。之后,北京师范大学信息科学与技术学院的研究人员,利用计算机技术成功复原了公主的头像。该项目组负责人周明全教授介绍,在西北大学的支持下,北京师范大学信息科学与技术学院虚拟现实与可视化中心的研究人员对公主的颅面通过CT和三维扫描建立人的皮肤、软组织、特征、五官四个数据库，通过数据库建立颅面的重构模型，随后对公主的颅面进行扫描,将获得的数据输入计算机,从数据库中找到和公主颅面特征相关的数据，对缺损部分给以修补,并以现代人头部数据为基础,采用回归统计法推导出颅面与面貌之间的关系，利用“计算机颅面复原软件”技术复原了公主的三维头像。
