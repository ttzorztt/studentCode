颅面复原技术

即三维颅面鉴定技术，属于国际法医学个体识别领域前沿领域。三维颅面复原技术是以颅面与面部关系为科学依据来生成三维人脸表面模型，是当前计算机图形学领域的一个研究热点，在医学、刑侦、考古等领域具有广泛的应用。
