颅面面貌复原技术的数据库收集和对比应用的过程。

法医技术
━━

 

 

 

 

 

       计算机辅助的二维颅面面貌复原方法：1987年George建立了新的颅面面貌技术,即颅面侧面描记法(lateral craniogaphic method),是依有明确软组织的头部侧面X线片测得正中线各测点的软组织厚度和颜面的各种角度,在未知颅面的侧面X线片描图上,重建近似的侧面像。1988年Perper等提出一种新的面貌复原方法,称为颜面影像复原形态图法(face imaging reconstruction morphography,FIRM)。其工作原理是利用自制的分析形态图(analytic morphography)记录颅面正位侧位X线片,然后再用计算机分析颅面的二维结构、几何学和比例的偏差,以此为依据确定面型,再根据脸型确定颊口鼻眼耳等五官。

       计算机辅助的二维颅面面貌复原方法:计算机辅助的二维颅面面貌复原由于其自身的限制,尽管不乏成功的案例,但是准确性一直无法超越面貌雕塑法。随着计算机技术的不断发展,出现了计算机辅助的三维颅面面貌复原方法。计算机辅助三维颅面面貌复原并不是手工颅面复原的计算机化,而是在发挥了计算机优点的基础上,引进模型和变形技术,使颅面面貌复原结果更为科学。

 

 

 

 

 

法医应用
━━

 

 

       重庆市公安局邢科所于70年代末开始将颅面面貌复原技术应用于刑事侦查工作中，成立由5人组成的固定研究班子，在学习古人类的复像经验基础上，不断在实践中改进和提高这项技术，采用的方法是无名死者颅面上沾泥土塑模，恢复其生前面貌形象，再以泥模为基础，用生石膏粉为原料，浇注翻造出石膏像。但是现在具有一个新的技术：计算机辅助的颅面面貌复原技术，发挥了计算机优点的基础上,引进模型和变形技术 ,使颅面面貌复原结果更为科学、更为快捷，此技术在刑侦方面取得了很多的成果，为警察抓取罪犯、判定死者相貌身份等都提供了帮助。

 

 

 

 

法医过程
━━

 

 

 

 

 

 

 

 

       为了最广泛收集不同的五官部件，采取不同人员的面部特征标志点，根据统计学原理，按照民族、性别、年龄进行分类统计。再从受试者中选择一批人，除了从十个不同角度拍照外，还需对其头部拍X光照片，将X光照片和本人照片进行比较，分析软组织和颅面之间的关系，是否和真人一致。最后将这些收集的颅面防砸专用设备内进行激光扫描，在电脑中形成三维图像。下一步，按照法医学指标，测量颅面的宽、高，眼眶内外间距，梨状孔宽和高，然后将这些数据和形态输入到计算机内，计算机以毫米为单位将面部分成不同区域，自动给颅面配上皮肤。接着启动五官数据库，相对应的眼睛、眉毛、鼻子、嘴、耳朵，会一一调取出来。比如根据颅面测量得出的嘴巴大概形状，数据库会列出十几个供选择，此时就需要专业人员进行分析，仔细观察颅面，选择一个最适合的安上去。一步就相当于往颅面上添肉、化妆，一张人脸图片就出现了。最后，还可以根据发现尸骨现场的情形，配上头发，穿上衣服等。

       发现无名颅面，无法确定死者身份和线索的情况下，将颅面送至专家处复原。将无名颅面，放在专用设备内进行激光扫描，电脑中形成无名颅面的三维图像。按照法医学指标，测量颅面的宽、高，眼眶内外间距，梨状孔宽和高，计算机将面部分成不同区域，自动给颅面配上皮肤。接着启动五官数据库，相对应的眼睛、眉毛、鼻子、嘴、耳朵，会一一调取出来。组装五官，相当于往颅面上添肉、化妆，一张人脸图片逐渐呈现出来。还可以根据发现尸骨现场的情形，配上头发。打印出照片并发布，由失踪人家属认出或提供线索，并且提供失踪人照片进行比对确认身份。

 

 

 

 

 

 

 

 

法医实例
━━

 

 

       2007年12月，江西省某县的一个石灰窑场石灰堆内发现一具尸骨，已完全白骨化。当地公安机关为查明尸骨身份，特将死者头骨送至广州市刑事科学技术研究所进行颅面面貌复原。袁中标将头骨模型数据输入其课题组自行研发的颅面测量三维复原系统，得出头骨性别、年龄、形态特征等重要判断数据和特征，并将其生前面貌恢复成三维照片供侦查使用。办案人员将复原照片拿回当地走访调查，群众反映，该县石灰窑场有一名女工巫某失踪已久，其面貌与复原像极为相似，相貌特征及形体结构也相吻合。结合DNA检验出相同结论的论证，所检颅面正是失踪人员巫某。袁中标主持研发的颅面测量三维复原系统，为该起案件指明真相方向、明确对象、固定证据起到了关键作用，案子得以迅速告破。
